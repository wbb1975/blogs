# Hadoop 数据类型与文件结构剖析 Sequence, Map, Set, Array, BloomMap Files
## 1. Hadoop’s SequenceFile
Hadoop的SequenceFile提供了对二进制键-值对的一种持久化数据结构，对比其它持久化键-值数据结构比如B树，你不能定位到特定键来编辑，添加或删除它，这种文件只能添加（append-only）。
